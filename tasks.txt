2. ООП
Дано: таблица в базе данных books (id, title, status), где статусы соответствуют следующим значениям: 0 - новая книга, 1 - опубликована на сайте, 2 - заблокирована, 3 - удалена.
Удаленные книги никто видеть не может, включая наш класс.
На сайте можно показывать только опубликованную книгу.
В админке можно показывать книги в статусах (0,1,2)
Нужно сделать класс работы с книгой, где можно
    1. взять одну книгу из базы (по id или title) с указанием источника (админка, сайт)
    2. добавить книгу (с указанием источника, из админки - публикуем сразу)
    3. изменить статус книги (только из админки)
